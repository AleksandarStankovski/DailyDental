VisualStudioCode tools:

1.Configuration:
- tsconfig.json
- launch.json
- tasks.json

2.Extensions:
- TSLint
- Typings auto install
- Debugger for Chrome
- Path intellisense

3.Angular 2 extensions:
- Angular language service // intellisense in templates
- Angular Snippets (angular-typescript-snippets or angular-snippets)
- Angular files (we can create angular cli with mouse right click)
- Type Lens (plugin which adds reference counter code lenses to typescript, js, scss and less)
- Types Auto Installer (any time you save either package.json or bower.json to typings packages will be installed/unistalled automatically in the background)
- Auto import
